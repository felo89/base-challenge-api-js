# Walmart challenge backend

## Instalation 
First install [database](https://github.com/walmartdigital/brand-discounts-db), Then:
```
npm install
```

## Start server

To run with node
```
npm start
```
To run in dev mode with nodemon
```
npm run dev
```

## Testing
To run all test run
```
npm run test
```
To run test with coverage run
```
npm run coverage
```
